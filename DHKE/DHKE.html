CSC 440/Cryptography
Diffie-Hellman Key Exchange

10 points

You and I are going to use the DHKE protocol to agree upon a secret key. Then you will bit-wise XOR that secret key with your seven-digit Depaul ID number as in integer (that is, a leading 0 is dropped). A seven-digit decimal number contains no more than 24 bits so the key will be 24 bits. Use my course notes on DHKE to see the function of the various variables. I will play the part of Alice and you the part of Bob. All values exchanged must be in decimal. The protocol will proceed as follows:

  1. We will use these public values: α = 1026, p = 33554467.
  2. I will select my secret value x and email each of you the integer that is the value of αx mod p.
  3. You will select your secret value y and compute z = αy mod p and the ciphertext integer c resulting from the bit-wise XOR of our agreed-upon key αxy mod p with your student id number. Put these into a text document with the three lines: 
    Your name
z as a decimal integer
c as a decimal integer
    For example: 
    John Rogers
12284838
3628334
  4. Using zx mod p I will decrypt c and verify that it is your id number. If not, I'll notify you and we'll try it again.

Submission instructions

Call the document DHKE.txt. Make sure it only contains the three lines specified above. Submit that file to the D2L submission folder provided for it.
